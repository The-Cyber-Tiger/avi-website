Tecnologías:

[ Proyecto ]

Es un asistente virtual desarrollado en lenguaje Python que funciona como interfaz entre el usuario y un brazo robot de uso industrial, integrando diversas tecnologías basadas en Inteligencia Artificial, el sistema potencia las capacidades del robot, facilitando la interacciónn del humano con los dispositivos y ofreciendo soluciones acorde a las necesidades de la Industria 4.0.

El proyecto fué reconocido en el XII Congreso Mexicano de Inteligencia Artificial y el artículo de investigación fué publicado por la revista internacional: “Research in Computing Science” Vol.149(8).

[ Interacción Humano-Máquina ]

Computer Vision

Mediante una cámara web acoplada al cabezal del robot el AVI es capaz de reconocer personas, objetos y colores aumentando considerablemente las capacidades de un brazo robot convencional.

Internet of Things

Dotado de conectividad WiFi el AVI permite el intercambio de información con cualquier dispositivo o sistema conectado a Internet.

Speech Recognition

El AVI cuenta con la capacidad de reconocer comandos de voz, igualmente de generar una respuesta como salida, interactuando de forma más intuitiva con el usuario.
